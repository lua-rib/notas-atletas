#  Cálculo da Média dos Atletas

Aplicação em **JavaScript** para calcular a média das notas de atletas em uma competição de ginástica artística.

##  Regras
- Cada atleta recebe **5 notas** (1 a 10)
- A **maior** e a **menor** nota são descartadas
- A média é calculada com as **3 notas restantes**

##  Saída esperada
- Nome do atleta
- Notas recebidas
- Média final

## Como executar
1. Salve o arquivo como `atletas.js`
2. Execute no terminal com:

```bash
node atletas.js
